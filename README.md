# HandParticleAI ✨

An interactive AI-powered hand tracking particle simulation built using Python, OpenCV, MediaPipe, and Pygame.

This project uses real-time computer vision to detect hand gestures and control dynamic particle physics effects.

---

# Features 🚀

- Real-time hand tracking
- AI-powered gesture detection
- Interactive particle system
- Pinch gesture particle explosion
- Neon glow particle effects
- Smooth particle physics
- Dynamic color animation
- Real-time graphics rendering

---

# Tech Stack 🛠️

- Python
- OpenCV
- MediaPipe
- Pygame
- Math & Physics Simulation

---

# Demo Concepts 🎮

- Move your index finger to attract particles
- Pinch fingers together to create explosions
- Watch particles dynamically react in real time

# Installation ⚡

## Clone Repository

```bash
git clone https://github.com/your-username/HandParticleAI.git
cd HandParticleAI
```

# Create Virtual Environment

## Windows
```bash
python -m venv venv
venv\Scripts\activate
```

# Install Dependencies

```bash
pip install opencv-python mediapipe pygame
```

# Run Project ▶️
```bash
python main.py
```
